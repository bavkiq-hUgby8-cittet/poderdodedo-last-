O Poder do Dedo

Regras do Jogo

Efeitos das Cartas:

  • A (Ás): O jogador escolhe alguém para beber uma dose.
  • 2: O jogador distribui duas doses para outras pessoas.
  • 3: O jogador distribui três doses para três pessoas diferentes.
  • 4: "Marca de..." O jogador escolhe uma categoria, e os jogadores devem falar uma marca dessa categoria até que alguém erre e tenha que beber.
  • 5: "Eu nunca..." O jogador fala uma afirmação sobre algo que nunca fez. Quem já fez, bebe uma dose.
  • 6: O jogador quebra todas as regras existentes.
  • 7: O jogador quebra uma regra existente.
  • 8: O jogador recebe o "Poder do Dedo" (pode ser ativado a qualquer momento).
  • 9: Inverte o sentido do jogo (horário/anti-horário).
  • 10: A vez passa para o jogador depois do próximo (pula um jogador).
  • J (Valete): O jogador que puxou a carta bebe uma dose.
  • Q (Dama): Todas as mulheres bebem uma dose.
  • K (Rei): Todos os homens bebem uma dose.
  • Coringa: O jogador ganha um coringa que pode ser usado para evitar beber.
